Running the program

The Python program make_projections.py will make the projections. Simply run this program with Python. It will ask you for the names of four files giving The input data; you must provide it with the names of the four files in the formats described in Input data file formats.

The input data

The model is built on vectors of data:

  • H0 is the vector of the Ha,‚ÄÖt‚ÄÖ=‚ÄÖ0 values, giving the number of HIV diagnosed individuals in our demographic category at year t‚ÄÖ=‚ÄÖ0 (whatever year we are using to start our projections) for each age category a .
  • M is the vector of the Ma of the mortality rate of individuals diagnosed with HIV in age category a in our demographic category.
  • D is the vector of the diagnosis rate Da of age group a in our demographic category.
  • Ct is the vector of the census counts of individual in our demographic category in age category a for year t . We have a vector provided by US Census data for each year t that we are modeling.
